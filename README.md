iHuza Inventory System

This is a React application that provides a user interface for managing inventory and product assignments in real-time.

Features:

Dashboard: Displays a summary of the inventory and product assignments.
Users: Manages user accounts and their roles.
Products: Manages product information and their assignments.
Assignments: Manages product assignments to users.
Categories: Manages product categories.
Authentication: Provides a login and logout functionality.
